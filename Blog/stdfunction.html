C++ Standard Library Functions and Algorithms

Common Member Functions for Containers:

  • size(): Returns the number of elements in the container.
  • empty(): Checks if the container is empty and returns a boolean value.
  • clear(): Removes all elements from the container.
  • insert(): Inserts elements into the container.
  • erase(): Removes elements from the container.
  • push_back(): Adds an element to the end of the container.
  • pop_back(): Removes the last element from the container.
  • emplace(): Constructs and inserts an element into the container.
  • find(): Searches for an element and returns an iterator to it.
  • count(): Counts the number of elements with a specific value.
  • lower_bound(): Returns an iterator pointing to the first element not less than the given key.
  • upper_bound(): Returns an iterator pointing to the first element greater than the given key.
  • equal_range(): Returns a pair of iterators denoting the range of elements with a specific key value.

Algorithms for Containers:

  • std::sort(): Sorts the elements in the container.
  • std::find(): Finds an element in the container and returns an iterator to it.
  • std::for_each(): Applies a function to each element in the container.
  • std::accumulate(): Computes the sum of elements in the container.
  • std::count(): Counts the occurrences of a specific value in the container.
  • std::max_element(): Finds the iterator pointing to the maximum element in the container.
  • std::min_element(): Finds the iterator pointing to the minimum element in the container.
