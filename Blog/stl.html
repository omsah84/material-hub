C++ Standard Library Components

Containers:

  • std::vector: Dynamic array.
  • std::list: Doubly-linked list.
  • std::deque: Double-ended queue.
  • std::set: Ordered set.
  • std::multiset: Ordered multiset(allows duplicate elements).
  • std::map: Ordered key-value pair container.
  • std::multimap: Ordered multimap.
  • std::unordered_set: Unordered set.
  • std::unordered_multiset: Unordered multiset.
  • std::unordered_map: Unordered map.
  • std::unordered_multimap: Unordered multimap.
  • std::stack: Stack data structure.
  • std::queue: Queue data structure.
  • std::priority_queue: Priority queue.

Algorithms:

  • std::sort: Sorts elements in a range.
  • std::binary_search: Performs binary search on a sorted range.
  • std::find: Finds an element in a range.
  • std::count: Counts occurrences of a value in a range.
  • std::for_each: Applies a function to each element in a range.
  • std::transform: Applies a function and stores the result in another range.
  • std::max_element: Finds the largest element in a range.
  • std::min_element: Finds the smallest element in a range.
  • std::accumulate: Computes the sum of elements in a range.
  • std::next_permutation: Generates the next lexicographically greater permutation of a range.
  • std::prev_permutation: Generates the next lexicographically smaller permutation of a range.

Iterators:

  • std::begin(): Returns an iterator pointing to the first element of a container.
  • std::end(): Returns an iterator pointing one past the last element of a container.
  • std::advance(): Moves an iterator by a specified number of steps.
  • std::distance(): Calculates the number of elements between two iterators.
  • std::next(): Returns an iterator advanced by a specified number of steps.
  • std::std::prev(): Returns an iterator moved back by a specified number of steps.

Utility:

  • std::pair: A class template that holds two values.
  • std::make_pair(): Creates a pair.
  • std::tuple: A class template that holds a collection of values.
  • std::make_tuple(): Creates a tuple.
  • std::swap(): Swaps the values of two variables.
  • std::move(): Casts an lvalue to an rvalue, enabling move semantics.
  • std::forward(): Casts an lvalue to an rvalue preserving its value category.
  • std::unique_ptr: Smart pointer for single ownership semantics.
  • std::shared_ptr: Smart pointer for shared ownership semantics.
  • std::weak_ptr: Smart pointer that does not change the reference count.

Other Utilities:

  • std::cin: Standard input stream.
  • std::cout: Standard output stream.
  • std::cerr: Standard error stream.
  • std::endl: Manipulator for inserting a newline character and flushing the stream.
  • std::cin.ignore(): Ignores characters in the input stream.
  • std::cin.getline(): Reads a line from the input stream.
  • std::cin.peek(): Reads a character without removing it from the input stream.
